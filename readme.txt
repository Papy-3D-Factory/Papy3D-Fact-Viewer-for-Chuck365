Displays a unique and different Chuck Norris fact every day via the official Chuck365.fr API.

== Description ==

Chuck365 Fact Viewer is a modern and lightweight plugin to add a touch of humor to your site. It retrieves a new Chuck Norris "fact" every day.

The plugin uses the latest WordPress technologies to guarantee optimal performance and maximum security.

== Installation ==

1. Download the plugin folder.

2. Upload it to the `/wp-content/plugins/` directory.

3. Activate the plugin via the WordPress 'Plugins' menu.

4. Add the "Chuck365 Fact Viewer" block or use the shortcode [chuck_fact].
